#  Fundamentos de Cloud Computing

## O que é Cloud Computing?

Cloud Computing (Computação em Nuvem) é a entrega de recursos de computação pela internet, incluindo servidores, armazenamento, bancos de dados, redes, software e análises.

##  Benefícios da Cloud

### 1. **Economia de Custos**
- Sem investimento inicial em hardware
- Paga apenas pelo que usa
- Redução de custos operacionais

### 2. **Escalabilidade**
- Aumenta ou diminui recursos conforme necessidade
- Elastic - se adapta à demanda

### 3. **Velocidade e Agilidade**
- Provisiona recursos em minutos
- Experimenta sem compromisso de longo prazo

### 4. **Alcance Global**
- Deploy em múltiplas regiões mundialmente
- Baixa latência para usuários globais

### 5. **Segurança**
- Infraestrutura protegida
- Compliance com padrões internacionais

##  Modelos de Serviço

### IaaS - Infrastructure as a Service
**O que é:** Aluguel de infraestrutura de TI

**Exemplos:** 
- Amazon EC2
- Google Compute Engine

**Você gerencia:** SO, aplicações, dados  
**Provedor gerencia:** Hardware, rede, virtualização

---

### PaaS - Platform as a Service
**O que é:** Plataforma para desenvolvimento

**Exemplos:**
- AWS Elastic Beanstalk
- Google App Engine

**Você gerencia:** Aplicações, dados  
**Provedor gerencia:** Runtime, SO, infraestrutura

---

### SaaS - Software as a Service
**O que é:** Software pronto para uso

**Exemplos:**
- Gmail
- Office 365
- Salesforce

**Você gerencia:** Apenas seus dados  
**Provedor gerencia:** Tudo

##  Modelos de Deploy

### 1. **Cloud Pública**
- Recursos compartilhados
- Menor custo
- Exemplo: AWS, Azure, GCP

### 2. **Cloud Privada**
- Recursos dedicados
- Maior controle e segurança
- Mais cara

### 3. **Cloud Híbrida**
- Combinação de pública e privada
- Flexibilidade
- Dados sensíveis na privada, demais na pública

##  Conceitos Importantes

**Elasticidade:** Capacidade de aumentar/diminuir recursos automaticamente

**Alta Disponibilidade:** Sistema continua funcionando mesmo com falhas

**Tolerância a Falhas:** Sistema se recupera automaticamente de falhas

**Disaster Recovery:** Plano para recuperação em caso de desastres
